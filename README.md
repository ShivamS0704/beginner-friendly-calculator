# Scientific Calculator in C

A feature-rich scientific calculator developed using the C programming language.  
This project combines basic arithmetic, scientific calculations, trigonometric functions, statistics, memory operations, and matrix computations within a simple console-based interface.

## Features

### Basic Calculator
- Addition
- Subtraction
- Multiplication
- Division

### Scientific Functions
- Square Root
- Power Calculations
- Natural Logarithm
- Log Base 10
- Exponential Functions
- Factorial

### Trigonometric Functions
- Sine
- Cosine
- Tangent

### Statistics Operations
- Mean
- Median
- Mode

### Matrix Operations
- Matrix Addition
- Matrix Subtraction
- Matrix Transpose
- Determinant Calculation (2x2 and 3x3)

### Memory Functions
- Memory Store (MS)
- Memory Recall (MR)

## Technologies Used
- C Programming Language
- Standard C Libraries
- Math Library (`math.h`)

## Concepts Used
- Functions
- Loops
- Conditional Statements
- Arrays
- Matrix Operations
- Mathematical Computations
- User Input Handling

## How to Run

### Compile
```bash
gcc calculator.c -o calculator -lm
```

### Run
```bash
./calculator
```

## Project Objective
The objective of this project is to strengthen programming fundamentals and problem-solving skills by implementing a multi-functional scientific calculator using C.
